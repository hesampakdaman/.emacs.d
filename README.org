* GNU Emacs
[[file:splash.gif]]

This configuration requires =Emacs 30.1+= to run. Usually, I build
Emacs from source with the following configuration
#+begin_src bash
  ./configure \
      --disable-gc-mark-trace \
      --with-dbus \
      --with-native-compilation \
      --with-pgtk \
      --with-tree-sitter \
      --with-x-toolkit=no \
      --without-compress-install \
      --without-xaw3d \
      CFLAGS="-O2 -march=native -mtune=native -fomit-frame-pointer"
#+end_src

After installing Emacs run the command
#+begin_src bash
  $ emacs --batch -l org ./init.org -f org-babel-tangle
#+end_src
which generates an =init.el= file that Emacs will use for
configuration. Now open up Emacs and it will automatically download
and install all declared packages. That's it!
